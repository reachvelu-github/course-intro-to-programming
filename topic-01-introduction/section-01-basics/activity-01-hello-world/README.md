# Activity 1: Hello World

## Problem Description

Write a program that prints "Hello, World!" to the console. This is the traditional first program that programmers write when learning a new language.

## Objectives

- Understand basic program structure
- Learn how to print output to the console
- Run your first program successfully

## Requirements

1. The output must be exactly: `Hello, World!`
2. The output should include a newline at the end
3. No extra spaces or characters

## Expected Output

```
Hello, World!
```

## Examples

### Example 1
**Input**: None
**Output**:
```
Hello, World!
```

## Instructions

1. Open the starter file for your chosen language
2. Write code to print "Hello, World!"
3. Run your program
4. Verify the output matches exactly

## Tips

- Pay attention to capitalization: "Hello" starts with capital H, "World" starts with capital W
- Don't forget the comma and exclamation mark
- Make sure there's a newline at the end

## Language-Specific Notes

### JavaScript
- Use `console.log()` to print to the console
- Strings can be enclosed in single or double quotes

### Python
- Use the `print()` function
- Strings can be enclosed in single or double quotes

### Java
- Use `System.out.println()` for output
- Strings must be enclosed in double quotes
- Don't forget the semicolon at the end
