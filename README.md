# Course: Introduction to Programming

This is a sample course repository demonstrating the structure required for the Course Activities VSCode extension.

## Course Overview

This course covers the basics of programming including:
- Variables and Data Types
- Control Flow
- Functions
- Basic Data Structures

## Prerequisites

- No prior programming experience required
- Computer with internet connection
- VSCode with Course Activities extension installed

## How to Use

1. Install the Course Activities extension in VSCode
2. Login with your GitHub credentials
3. Browse topics and activities in the tree view
4. Select your preferred programming language
5. Complete activities in order

## Course Structure

- **Topic 1: Introduction** - Getting started with programming
- **Topic 2: Variables** - Understanding variables and data types
- **Topic 3: Control Flow** - Conditionals and loops

## Supported Languages

- JavaScript
- Python
- Java
